# Control Center Widget
Change screen brightness or system volume with easy controls.

# Available controls
* Brightness (-/+ and slider-toggle)
* Volume (-/+ and slider-toggle)
* Sleep
* Screensaver
* Lock Screen
* Do Not Disturb
